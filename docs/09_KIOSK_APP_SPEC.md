# 09 — Customer Kiosk App Spec

## 1. Purpose

Customer browses visually on tablet.

## 2. Home

```text
BÁC CẦN MUA GÌ?

[ Cám chăn nuôi ]
[ Phân bón ]
[ Thuốc sâu / thuốc bệnh ]
[ Thuốc cỏ ]
[ Thuốc chuột ]
[ Hạt giống ]
[ Hỏi trợ lý ]
[ Gọi người bán ]
```

## 3. Product card

- big image
- display name
- price
- short use
- stock status
- view/select buttons

## 4. Wanted list

Customer selects products, gets a code.

Staff retrieves and creates POS sale.

## 5. Security

No login as admin on kiosk.

No private fields.

## 6. Chemical warning

Always show safety note for chemical products.
