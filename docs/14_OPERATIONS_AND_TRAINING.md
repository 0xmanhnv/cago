# 14 — Operations and Training

## 1. Owner rollout

Week 1:

- Tra giá only.

Week 2:

- Sửa giá.

Week 3:

- Ghi nợ / trả nợ.

Week 4:

- POS/bán hàng.

## 2. Training phrase

Do not say "ERPNext".

Say:

```text
Mẹ chỉ cần dùng mấy nút này:
Tra giá, Sửa giá, Ghi nợ, Khách trả nợ, Báo cáo hôm nay.
```

## 3. Daily routine

Morning:

- check low stock
- check price changes

During day:

- tra giá
- sell through POS
- record debt

End day:

- today's report
- debt list

## 4. Backups

- daily DB backup
- daily files/images backup
- weekly offsite copy
- monthly restore test

## 5. Tablet

- kiosk mode
- no admin login
- stable Wi-Fi
- keep charged
